Java常用类

String

  • String是一个final类，不可继承
  • 字符串的内容在String对象中以一个char型数组value[]保存，该属性由final修饰，不可改变
  • 双引号引起来的部分是常量，存储在方法区的字符串常量池中，是不可修改的
  • 常量与常量拼接的结果在常量池，常量池中不会存在相同内容的常量
  • 字符串拼接时，只要有一个是变量的形式，拼接的结果就是在堆中，若结果调用intern()方法，其返回值在常量池中
常用方法
int length()//返回字符串长度
int indexOf(String str)//返回指定字符串在当前字符串中首次出现的索引，没有返回-1
int indexOf(String str, int fromIndex)//返回从指定索引位置开始指定字符串在当前字符串中首次出现的索引
int lastIndexOf(String str)//从右往左查找
int lastIndexOf(String str, int fromIndex)//从指定索引往左查找！
int compareTo(String anotherString)//比较两个字符串的大小(逐个比较字符的值)
char charAt(int index)//返回指定索引位置的字符
boolean isEmpty()//判断字符串是否为空
boolean endsWith(String suffix)//判断是否以指定字符串结尾
boolean startsWith(String prefix)//判断是否以指定字符串开始
boolean startsWith(String prefix, int toffset)//判断从指定索引位置起是否以指定字符串开始
boolean contains(CharSequence s)//判断字符串是否包含指定char值序列
boolean equals(Object obj)//比较字符串内容是否相同
boolean equalsIgnoreCase(String anotherString)//与equals()类似，忽略大小写
boolean matches(String regex)//判断字符串是否匹配给定正则表达式

String toLowerCase()//转换String中的字符为小写
String toUperCase()//转换String中的字符为大写
String trim()//去掉首位空白的字符串副本
String concat(String str)//字符串拼接，作用相当于 +
String replace(char oldChar, char newChar)//将字符串中所有oldChar换为newChar
String replace(CharSequence target, CharSequence replacement)
String replaceAll(String regex, String replacement)//用replacement替换所有正则表达式(regex)匹配项
String replaceFirst(String regex, String replacement)//用replacement替换第一个正则表达式(regex)匹配项
String substring(int beginIndex)//截取从beginIndex到最后的字符串
String subString(int beginIndex, int endIndex)//截取从beginIndex到endIndex(不包括)的字符串
String[] split(String regex)//根据指定正则表达式拆分字符串
String[] split(String regex, int limit)//根据指定正则表达式，拆分为最多limit个字符串
类型转换

  • 字符串 → 基本数据类型、包装类

    • 纯数字字符串转换为整型：Integer.parseInt(String str)
    • 其他基本数据类型类似（Byte、Short、Long、Float、Double）

  • 基本数据类型、包装类 → 字符串

    • String.valueOf(int n)
    • 其他类似：valueOf(byte b)、valueOf(long l)、valueOf(float f)、valueOf(double d)、valueOf(boolean b)

  • 字符串 → 字符数组

    • char[] toCharArray()
    • void getChars(int srcBegin, int srcEnd, char[] dst, int dstBegin): 将指定索引范围的字符串存放到指定数组中

  • 字符数组 → 字符串

    • String类构造器：String(char value[])、String(char value[], int offset, int count)等

  • 字符串 → 字节数组

    • byte[] getBytes(): 使用默认字符集将字符串序列编码为byte序列
    • byte[] getBytes(String charsetName): 使用指定字符集(utf-8、gbk等)将字符串序列编码为byte序列

  • 字节数组 → 字符串

    • String类构造器：String(byte bytes[])、String(byte bytes[], int offset, int length)等

StringBuffer

  • 可变的字符序列
  • 属性 char[] value 没用final声明，可以不断扩容
  • 属性count记录有效字符个数
  • 必须用构造器生成
    • StringBuffer(): 初始容量为16的字符串缓冲区
    • StringBuffer(int size): 指定初始容量
    • StringBuffer(String str): 将传入字符串初始化
常用方法
StringBuffer append(...)//用于进行字符串拼接，有很多重构的append()方法
StringBuffer delete(int start,int end)//删除指定索引范围的内容
StringBuffer replace(int start, int end, String str)//把[start,end)范围的内容替换为str
StringBuffer insert(int offset, xxx)//在指定位置插入xxx
StringBuffer reverse()//把当前字符序列逆转

//许多方法与String相类似
int length()
int indexOf(String str)
char charAt(int n )
String substring(int start,int end)
void setCharAt(int n ,char ch)

StringBuilder

  • StringBuilder 和 StringBuffer 非常类似，均代表可变的字符序列，而且提供相关功能的方法也一样
  • 对比String、StringBuffer、StringBuilder
    • String(JDK1.0)：不可变字符序列
    • StringBuffer(JDK1.0)：可变字符序列、效率低、线程安全
    • StringBuilder(JDK 5.0)：可变字符序列、效率高、线程不安全
  • 注意：作为参数传递的话，方法内部String不会改变其值，StringBuffer和StringBuilder 会改变其值。
